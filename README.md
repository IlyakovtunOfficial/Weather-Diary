Weather Diary
Имя и фамилия автора: Ковтун Илья Вячеславович
Краткое описание программы: Weather Diary — простое GUI-приложение для записи дневника погоды. Можно добавлять записи, просматривать их в таблице, фильтровать по дате и температуре, сохранять и загружать данные в формате JSON, а также хранить проект в Git/GitHub.
Как запустить приложение:
Установите Python 3.x.
Сохраните weather_diary.py и запустите: python weather_diary.py
Как запустить с Git:
git clone https://github.com/ваш_пользователь/weather-diary.git
cd weather-diary
python weather_diary.py
Примеры использования:
Введите дату: 2026-05-01, температуру 15.5, описание: "Пасмурно, лёгкий ветер", осадки: да > нажмите "Добавить запись".
В фильтре задайте дату: 2026-05-01 > приложение отобразит записи за эту дату.
В фильтре минимальная температура: 10 > будут показаны записи с температурой >= 10°C.
Пример структуры проекта и заметки
weather_diary.py — основной код, GUI и логика.
weather_records.json — файл-данные (пример, можно создавать вручную или через приложение).
README.md — документация.
.gitignore — список файлов и директорий, которые не нужно хранить в репозитории.
requirements.txt — если вы добавляете зависимости (в базовом примере Tkinter не требует зависимости).
